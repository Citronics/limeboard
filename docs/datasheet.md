# Limeboard – Technical Specifications

**Product:** Limeboard  
**Document type:** Technical specification (informational)  
**Audience:** Engineering / Integration  
**Last updated:** 23/12/2025

---

## 1. Processing

- 1× Quad-core 32-bit Qualcomm Krait 400 CPU (ARMv7-A compatible), up to 2.26 GHz
- 1× 32-bit ARM Cortex-M4 microcontroller (STM32F401), up to 84 MHz
- 1× Adreno 330 GPU, up to 578 MHz

---

## 2. Memory and Storage

- 2 GB RAM
- 32 GB eMMC
- External SD / MMC interface

---

## 3. Wireless Connectivity

- Cellular: 2G / 3G / 4G LTE Cat. 4  
  - Dual µSIM  
  - Requires external antennas (not included)
- Wi-Fi: IEEE 802.11 b/g/n/ac (2.4 GHz / 5 GHz)
- Bluetooth: Bluetooth 4.0

---

## 4. Wired Communication and I/O

- 4× USB 2.0  
  - 3× data + power  
  - 1× power only
- 1× Ethernet 10/100 Mbps
- 1× 40-pin GPIO header (HAT compatible)  
  - UART, SPI, I²C, GPIO

---

## 5. Sensors

- GNSS / GPS, A-GPS  
  - Requires external antenna (not included)
- Accelerometer  
  - Software support pending

---

## 6. Power

- Input voltage: 5 V – 60 V
- Typical power consumption: 2 W – 20 W (depending on configuration and usage)
- USB-C Power Delivery (PD)
- Power over Ethernet (PoE)

---

## 7. Mechanical

- Dimensions: 142 × 70 × 24 mm
- Weight: 95 g
